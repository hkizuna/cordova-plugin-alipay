# cordova-plugin-alipay

A cordova plugin, a JS version of Alipay SDK

# Install

1. ```cordova plugin add https://github.com/hkizuna/cordova-plugin-alipay --variable partnerid=your_partnerid```

2. ```cordova build ios```
